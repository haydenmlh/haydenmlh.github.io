Personal Website Inspiration and Process
Photo by James Harrison on Unsplash

The primary purpose of my personal website is to showcase my programming projects and photography. To achieve this, I began the website design by splitting my navigation into several subsections, namely Home, About, Projects, Photography, Blog and Contact. After browsing an article on Canva about website color schemes, I decided to incorporate the No.11 Elegant and Sophisticated colour scheme from the article for my website, as seen on the right.

I then proceeded to design the navigation bar and make it responsive for mobile and desktop. Through designing the responsive part for mobile, I learnt about how to add metadata to the head to make the website display on mobile properly. I also resized different elements through CSS and Javascript to maneuver them into an aesthetically pleasing layout. The process involved a lot of using the latest CSS3 technologies including flex and grid display, as well as dynamic resizing based on viewport size using Javascript.

After seeking out constructive criticism from close friends, I changed the look and colors of the website for a cleaner, more minimalistic style.
